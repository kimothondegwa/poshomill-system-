## ✋ One More Step (If Using VirtualHost)

If you want to use `http://poshomill.local`, edit your hosts file:

**Windows 10/11:**
1. Right-click **Notepad** → **Run as Administrator**
2. File → Open → `C:\Windows\System32\drivers\etc\hosts`
3. Add at the end:
   ```
   127.0.0.1       poshomill.local
   ```
4. Save and close

Then restart Apache in XAMPP Control Panel.
